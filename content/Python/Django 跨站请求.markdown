通过两个装饰器, 可以对 Django 中的函数强制开启跨站请求伪造功能:
@csrf_protect, 为当前函数强制设置防跨站请求伪造功能, 即便 settings.py 中没有设置全局中间件;
@csrf_exempt, 取消当前函数防跨站请求伪造功能, 即便 settings.oy 中设置了全局中间件;
